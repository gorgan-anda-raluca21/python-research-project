# EEG Depression Detection using Python

## 📌 Description

This project focuses on analyzing EEG signals to detect depression using data analysis and machine learning techniques. The workflow includes data exploration, preprocessing, feature handling, and classification.

## ⚙️ Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## 🧠 Methodology

The project follows these main steps:

1. **Data Loading & Exploration**

   * Loading EEG dataset from CSV files
   * Inspecting structure, features, and distributions

2. **Data Preprocessing**

   * Handling missing values (if applicable)
   * Selecting relevant features
   * Preparing data for modeling

3. **Machine Learning Models**

   * Decision Tree classifier
   * Model training and prediction

4. **Model Evaluation**

   * Confusion Matrix for performance analysis
   * Accuracy and classification insights

## 📊 Results

The project demonstrates how EEG-based features can be used to distinguish between depressive and non-depressive cases. The confusion matrix and model outputs provide insights into classification performance.


## ▶️ How to Run

1. Open the notebook:
   `eeg-depression-detection.ipynb`
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib scikit-learn
   ```
3. Run all cells in order
